# Volumes and Files — kOS 1.4.0.0 Documentation

## Overview

This documentation page serves as an index for structures related to file and volume management in kOS. The page lists six primary components within the "Volumes and files" category.

## Structure Components

The documentation covers the following structures:

- **FileContent** — handles file content operations
- **Path** — manages file system paths
- **Volume** — represents storage volumes
- **VolumeDirectory** — directory management within volumes
- **VolumeFile** — individual file representation
- **VolumeItem** — generic volume item base structure
