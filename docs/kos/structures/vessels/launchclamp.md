# LaunchClamp

`LaunchClamp` is provided to help identify launch clamps among parts and extends the `Decoupler` type, which itself extends `Part`.

## Structure: LAUNCHCLAMP

| Suffix | Type | Description |
|--------|------|-------------|
| All suffixes of `Decoupler` | — | A `LaunchClamp` inherits all capabilities from `Decoupler` (which inherits from `Part`) |

## Inheritance Hierarchy

`LaunchClamp` is a specialized type of `Decoupler`, which is itself a `Part`. This means all properties and methods available on those parent types are accessible on `LaunchClamp` instances.
