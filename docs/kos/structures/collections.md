# Collections

The Collections section of the kOS 1.4.0.0 documentation covers data structure types used for organizing and managing groups of elements.

## Available Collection Types

The documentation outlines eight primary collection structures:

- **Enumerable** — provides iteration capabilities
- **Iterator** — enables sequential access through collections
- **Lexicon** — key-value pair storage (dictionary-like)
- **List** — ordered sequence of elements
- **Queue** — first-in, first-out data structure
- **Range** — sequence of numbers within a range
- **Stack** — last-in, first-out data structure
- **UniqueSet** — collection of unique values

## Documentation Organization

This page serves as an index within the broader kOS Structure Reference section. Each collection type links to detailed documentation covering its specific methods, properties, and usage patterns within kOS scripting.

The documentation is part of the kOS project, maintained by the kOS Team since 2013, and built using Sphinx documentation tools.
