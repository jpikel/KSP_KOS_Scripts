# Making GUIs in kOS

## Overview

The kOS documentation describes GUI creation as a feature introduced in version 1.1.0. According to the page, "Sometimes it is useful to create an in-game set of screen widgets to let you interact with the kOS program with mouseclicks."

## Key Points

**Functionality**: Users can create GUI widgets in kOS with varying complexity levels. These widgets operate within pop-up windows that appear during program execution.

**Important Limitation**: The documentation notes that "these widgets cannot be operated via the telnet feature. You need to manipulate them with the mouse on-screen in the game window."

**Starting Point**: The primary object type for GUI creation is the `GUI` structure, which is documented separately in the structures section of the documentation.

## Documentation Structure

The page serves as an introductory overview, with the detailed technical information about GUI widget implementation referenced on a separate dedicated page within the documentation set.
