# Community Examples Library

## Overview

"Starting with version 0.17.0 of kOS, we have decided to support a separate repository of examples and libraries that 'live' entirely in Kerboscript code only."

This resource serves as a collection space for helpful code written by kOS users and developers.

## Repository Location

The separate repository can be found at:
[https://github.com/KSP-KOS/KSLib](https://github.com/KSP-KOS/KSLib)

## Available Resources

Users can discover various useful tools in this repository, including:

- A library for obtaining **navball orientation** information
- An example demonstrating the sasmode feature implementation
- Routines for enumerating over lists, queues, and stacks using callback delegates
- A seven-segment display controller
- A keypad emulator enabling visual keyboard navigation and text input
- A library for rendering chunky lines using terminal character cells
- A routine calculating optimal launch timing to achieve target inclination orbits
