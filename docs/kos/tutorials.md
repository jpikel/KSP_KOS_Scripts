# Tutorials

## Overview

This documentation page introduces the tutorial resources available for kOS 1.4.0.0. The tutorials are organized by skill level and provide "walk you through examples" for learning the system.

## Introductory Tutorials

**Quick Start Tutorial**

"Walks you through the beginnings of making a beginner's ship launcher script."

**Basic Explanation Tutorial**

This resource covers foundational kOS commands.

## Intermediate Tutorials

**Design Patterns Tutorial**

This tutorial "discusses some general aspects of kOS flow control and optimizations."

**PID Loop Tutorial**

"Starts with a basic proportional feedback loop and develops, in stages, a complete PID-loop to control the throttle of a simple rocket design."

**Execute Node Script**

ZiwKerman presents a generic maneuver node execution script described as "a one-size-fits-all solution to many situations in KSP."

**Display Bounds Script**

An example program demonstrating vessel and part bounding box reading, vector rotation transformation, and VECDRAW line drawing capabilities.

**Creating Reusable GUI Elements**

This tutorial focuses on building and implementing a "TabWidget" for condensing functionality into limited screen space.

## External Resources

The documentation notes that "tutorials written by others, outside the main kOS docs" are available, including general tutorials maintained separately on GitHub.
