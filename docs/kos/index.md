# kOS: Kerbal Operating System Documentation

## Overview

kOS is a community-supported mod for Kerbal Space Program described as "the do-it-yourself autopilot." The documentation introduces it as "an autopilot you script yourself," designed to scale with user skill levels from simple to complex spacecraft operations.

## Core Functionality

The mod introduces virtual computer parts that run programs in **kerboscript**, kOS's scripting language. These simulated computers execute within the game's Unity engine rather than on the player's computer, maintaining the in-game universe concept.

## Getting Started

New users are directed to a quickstart tutorial featuring a simple launching script. Installation follows standard mod procedures: extract contents into the Kerbal Space Program folder, installing into the GameData directory.

## Community and History

kOS maintains an active user community for sharing ideas and algorithms. The project was originally created by Kevin Laity (Nivekk) and has since undergone significant architectural changes under continued development by a larger team.
