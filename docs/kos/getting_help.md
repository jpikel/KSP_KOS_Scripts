# Getting Help

Although documentation strives for clarity, questions inevitably arise. The kOS project offers several support channels:

## Reddit

The [kOS subreddit](https://www.reddit.com/r/Kos/) serves users well for inquiries requiring detailed discussion. It works best when you need to share scripts for troubleshooting or seek inspiration from community code examples.

## Discord

The [Discord server](https://discord.gg/Hjsz7r9) provides a more immediate communication platform suitable for quick questions and general conversations about the mod.

## Github Issues

The [GitHub issues page](https://github.com/KSP-KOS/KOS/issues) is appropriate for reporting bugs, asking general project questions, or proposing contributions.
