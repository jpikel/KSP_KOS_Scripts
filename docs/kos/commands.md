# Command Reference — kOS 1.4.0.0

This documentation page serves as an index for kOS commands and features. Rather than detailed content, it presents a hierarchical navigation structure organized into several major categories:

## Overview

The page provides access to command documentation across multiple domains:

- **Running Programs**: Program execution functions and the RUN keyword
- **Flight Control**: Includes cooked control, raw control, pilot input, and ship systems
- **Prediction**: Orbital and trajectory prediction features
- **Listing Data**: FOR loops and available listable keywords
- **Parts Information**: Part-related commands and data
- **File I/O**: Directory structures, paths, volumes, file operations, and JSON handling
- **Terminal and GUI**: Display tools and interface elements
- **Serialization**: Data serialization features
- **Communication**: Message passing, queues, connections, vessel and processor communication
- **Resource Transfer**: Resource management and transfer operations

## Structure

The page functions primarily as a documentation hub with links to detailed sections rather than presenting extensive command examples or code. Each category branches into subcategories with dedicated documentation pages.

The documentation is built using Sphinx and maintained by the kOS Team (originally by Nivekk), with copyright held from 2013-2021.
