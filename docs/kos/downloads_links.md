# Downloads and Video Links

## Obtaining kOS itself

The kOS mod can be acquired through several distribution channels:

- **CKAN**: The Comprehensive Kerbal Archive Network allows automated mod management and update tracking. "kOS sends update information to CKAN so kOS updates should appear in your CKAN manager tool as soon as they're released."

- **GitHub**: Development happens at the official repository, where "new releases will apear there first, without delays, and its where old obsolete releases can be found."

- **SpaceDock**: A community-supported hosting platform described as "the successor to KerbalStuff," featuring mod distribution without intrusive advertising.

- **Curse**: Listed as where SQUAD directs players for KSP addons, making it an official distribution point.

## Change Log

The complete change log is maintained on GitHub at the project repository.

## Online Community

Two primary discussion venues exist for kOS users:

- **/r/kos on Reddit**: Supports multiple discussion threads about the mod
- **KSP Official Forums**: Houses unified kOS discussions in a single thread

"The Reddit one tends to be more active because it allows multiple sub-threads about kOS."
