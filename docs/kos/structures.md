# kOS 1.4.0.0 Structure Reference

The provided page is a documentation index for the kOS scripting language's structure reference. Rather than reproduce the entire navigation menu and links, here's a summary of its key content:

## Overview

The page serves as a comprehensive guide to kOS structures, with a note that "a general discussion of structures can be found" in the language features section.

## Main Categories

The documentation organizes structures into these major sections:

- **Generic Structure** (Reflection)
- **Collections** (Enumerable, Iterator, Lexicon, List, Queue, Range, Stack, UniqueSet)
- **Volumes & Files** (FileContent, Path, Volume, VolumeDirectory, VolumeFile, VolumeItem)
- **Vessels & Parts** (40+ structure types including Vessel, Engine, Part, DockingPort, etc.)
- **Communication** (Connection, Message, MessageQueue)
- **Orbits** (Orbit, OrbitEta, Orbitable, OrbitableVelocity)
- **Celestial Bodies** (Atmosphere, Body)
- **GUI Widgets**
- **Miscellaneous** (Boolean, Colors, PIDLoop, String, Terminal, Time, Voice, etc.)
- **Waypoints**

The page is built with Sphinx documentation tools and is maintained by the kOS Team.
