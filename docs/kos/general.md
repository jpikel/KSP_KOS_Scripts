# kOS 1.4.0.0 General Topics Documentation

This page serves as an index for general topics covering the interface between kOS and Kerbal Space Program.

## Overview

The documentation organizes general topics into several key categories:

**Bound Variables and Core Systems**
- Named vessels and bodies, constants, terminal access, core functionality, and archive management
- Stage management, maneuver nodes, and resource types
- Flight control, time tracking, and system variables

**Hardware and Performance**
- CPU vessel configuration
- Hardware specifications including update ticks, physics ticks, electric drain, and triggers
- CPU update loop mechanics and the "frozen universe" concept

**Boot and Startup**
- Boot file configuration and execution timing
- Launchpad spawn behavior

**Game Integration**
- kOS part module with disk space, mass, and power consumption parameters
- Settings windows for control panel and difficulty configurations
- User GUI creation tools

**Data Management and I/O**
- File storage, volumes, and the archive system
- Boot directory special handling
- Machine code compilation to KSM files

**Hardware Features**
- Sound chip (SKID) capabilities
- Part naming tags and cloning with symmetry
- Parts and PartModules interaction

**Career Mode**
- Rules enforcement for career-limited features

Each topic links to detailed documentation pages covering implementation specifics, examples, and configuration options.
